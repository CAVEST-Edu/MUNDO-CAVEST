Matemática - Funções

1. O que é uma função?

Uma relação que associa cada elemento do domínio a exatamente um elemento do contradomínio
Qualquer equação matemática
Apenas gráficos de linha
Um número específico

Resolução:

Uma função é uma relação que associa cada elemento do domínio a exatamente um elemento do contradomínio.

Resposta correta: A

2. Qual é o domínio da função f(x) = √(x-2)?

Todos os números reais
x ≥ 2
x ≤ 2
x > 0

Resolução:

Para que a função f(x) = √(x-2) seja definida, o valor dentro da raiz deve ser ≥ 0, ou seja: x-2 ≥ 0 → x ≥ 2.

Resposta correta: B

3. Qual das alternativas representa uma função linear?

f(x) = 2x + 5
f(x) = x² + 3
f(x) = √x
f(x) = 1/x

Resolução:

Uma função linear tem a forma f(x) = mx + b, onde m e b são constantes. Aqui, f(x) = 2x + 5 é linear.

Resposta correta: A
