1. Qual é o valor de log₁₀ 1000?

log₁₀ 1000 = x → 10ˣ = 1000 → x = 3

2. Qual é a propriedade correta de logaritmos?

log(a × b) = log a + log b
log(a + b) = log a + log b
log(a - b) = log a - log b
log(a ÷ b) = log a × log b

Resolução:

A propriedade correta é: log(a × b) = log a + log b

Resposta correta: A
