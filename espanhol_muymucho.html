Español - Uso de Muy y Mucho ⚖️

Muy: Se usa antes de Adjetivos y Adverbios.
(Ej: Muy inteligente, Muy tarde).

Mucho: Se usa con Sustantivos o después de Verbos.
(Ej: Mucho dinero, Trabajo mucho).

1. Completa la frase correctamente:
"Ella está _______ cansada porque trabajó _______ hoy."

mucho - muy
muy - mucho
muy - muy
mucho - mucho

Resolución:

1. "Cansada" es un adjetivo, por eso usamos Muy.
2. "Trabajó" es un verbo, y después de verbos usamos Mucho.

Resposta correta: B

2. ¿Cuál de las siguientes frases es INCORRECTA?

Hablas muy bien español.
Tengo mucho hambre.
El examen fue muy difícil.
Mi casa es mucho grande.

Resolución:

"Grande" es un adjetivo. Por lo tanto, lo correcto sería decir "Muy grande". La opción D usa "mucho" de forma incorrecta.

Resposta correta: D

3. Existen excepciones. Usamos MUCHO antes de "mejor", "peor", "mayor" y "menor". Entonces, lo correcto es:

Mi hermano es muy mayor que yo.
Este libro es muy mejor que el otro.
Tu nota es mucho mejor ahora.
Hoy me siento muy peor.

Resolución:

Aunque "mejor" parece un adjetivo que debería llevar "muy", es una de las 4 excepciones principales. Siempre decimos Mucho mejor, Mucho peor, Mucho mayor y Mucho menor.

Resposta correta: C
