English - Conjunctions & Linkers 🔗

1. In the sentence: "She studied very hard; however, she didn't pass the exam.", the highlighted linker expresses:

Addition (Adição)
Contrast (Contraste)
Conclusion (Conclusão)
Reason (Causa)

Resolução:

A palavra "however" (no entanto, porém) é um conectivo de contraste ou oposição. Ela liga duas ideias opostas: estudou muito mas não passou.

Resposta correta: B

2. Choose the correct linker to complete the sentence:
"I'm staying home _______ it's raining heavily."

but
although
because
so

Resolução:

A frase indica uma razão ou causa para ficar em casa. "Because" (porque) é o linker ideal para introduzir uma explicação.

Resposta correta: C

3. Which of these linkers is used to add more information (Addition)?

Furthermore
Despite
Therefore
Otherwise

Resolução:

"Furthermore" (além disso/além do mais) é usado para adicionar informações a um argumento. "Despite" é contraste, "Therefore" é conclusão e "Otherwise" é condição/alternativa.

Resposta correta: A
