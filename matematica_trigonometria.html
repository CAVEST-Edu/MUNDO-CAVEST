Matemática - Trigonometria

1. Qual é o valor de sen(90°)?

0
1
0,5
-1

Resolução:

O seno de 90° é 1.

Resposta correta: B

2. Qual é a relação trigonométrica de um triângulo retângulo para tangente?

cateto adjacente / hipotenusa
cateto oposto / hipotenusa
cateto oposto / cateto adjacente
hipotenusa / cateto oposto

Resolução:

A tangente de um ângulo em um triângulo retângulo é a razão entre o cateto oposto e o cateto adjacente:
tan θ = cateto oposto / cateto adjacente.

Resposta correta: C

3. Se cos(θ) = 0,5 e θ está no primeiro quadrante, qual é o valor de θ?

30°
45°
60°
90°

Resolução:

No primeiro quadrante, se cos(θ) = 0,5, então θ = 60°.

Resposta correta: C
